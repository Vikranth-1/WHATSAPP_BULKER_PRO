# WhatsApp Bulker Pro

**WhatsApp Bulker Pro** is a premium, desktop-based automation tool designed to send personalized messages to multiple WhatsApp contacts with ease. Featuring a modern glassmorphism interface and a dedicated desktop experience, it makes bulk messaging both professional and efficient.

---

## Features

*   **Premium Glassmorphism UI**: A sleek, dark-themed dashboard with animated background blobs and backdrop filters for a state-of-the-art look.
*   **Standalone Desktop App**: No need for a separate browser window to manage configurations. Runs as a native Windows application.
*   **Smart Random Delays**: Built-in 8-15 second random delays between messages to mimic human behavior and protect your account from being flagged.
*   **Real-Time Live Logs**: A dedicated activity window that streams live updates from the automation engine.
*   **Session Persistence**: Safely remembers your WhatsApp login so you don't have to scan the QR code every time you open the app.
*   **Easy Configuration**: Simple inputs for phone numbers and messages directly in the app.

---

## How to Use

1.  **Download & Run**: Download the `WhatsAppBulker.exe` from the [releases](#) page.
2.  **Launch Browser**: Click the **"Launch Browser"** button. A Chrome instance will open to `web.whatsapp.com`.
3.  **Scan QR Code**: Scan the QR code with your phone to log in to WhatsApp Web.
4.  **Configure**:
    *   Enter your target phone numbers (one per line, including country codes).
    *   Type your message in the message box.
    *   (Optional) Adjust the Min/Max delay settings.
5.  **Start Automating**: Click **"Start Sending"** and watch the live logs!

---

## Prerequisites

*   **Google Chrome**: You must have the Google Chrome browser installed on your machine.
*   **Windows OS**: Currently optimized for Windows 10/11.

---

## Disclaimer

This tool is intended for **legitimate communication purposes** (e.g., event reminders, community updates). Automating WhatsApp can lead to account bans if used for spamming. Always ensure you have consent from the recipients.
